## chatcalc
chatcalc is an android app that disguises itself as a calculator but hides a secret way to talk to your friends (or yk 😏) through Discord and Instagram.

On first setup, you will be prompted to set a secret PIN which you can then use to unlock the chat functionallity.

## Features
- Completely stealth chats
- Discord + Instagram web wrapped
- Switches back to calc mode when closed
- Hidden in recents

## Installation
Download and install the APK from the releases section

Enable installation from unknown sources if prompted
